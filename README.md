# Task Tracker CLI

Project URL: https://roadmap.sh/projects/task-tracker

A simple command-line interface to manage your tasks, built with Node.js. This project was built as part of the roadmap.sh backend challenges.

## How to Use
1. **Add a task:** `node task-cli.js add "Task description"`
2. **List all tasks:** `node task-cli.js list`
3. **Mark as done:** `node task-cli.js mark-done 1`
